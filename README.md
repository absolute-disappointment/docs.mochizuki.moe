## Development

### Requirements

- Python 3.x

### Setup

```bash
$ pipenv install
```

### Start Server

```bash
$ pipenv run mkdocs serve
```

## Deploy to Github pages

```
$ pipenv run ./build.sh
$ ./deploy.sh
```
